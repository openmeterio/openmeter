# models

<!-- archie:ai-start -->

> Auto-generated Python model classes and enums for the entire OpenMeter API surface. Every request/response body type and every string enum is defined here; this is the single source of truth for Python-typed API objects in the SDK.

## Patterns

**rest_field attribute declaration** — Each field is declared as a class variable using rest_field(name='camelCaseName', visibility=[...]) to map Python snake_case to JSON camelCase. Required fields have no default; optional fields use Optional[T] = rest_field(...). (`phone_number: Optional[str] = rest_field(name='phoneNumber', visibility=['read', 'create', 'update', 'delete', 'query'])`)
**Dual __init__ overload pattern** — Every model class defines two @overload signatures: one with keyword args for IDE completion, one with mapping: Mapping[str, Any] for raw-dict construction. The real __init__ calls super().__init__(*args, **kwargs). (`@overload def __init__(self, *, field: str) -> None: ...
@overload def __init__(self, mapping: Mapping[str, Any]) -> None: ...
def __init__(self, *args, **kwargs) -> None: super().__init__(*args, **kwargs)`)
**CaseInsensitiveEnumMeta for all enums** — All enums inherit from (str, Enum) with metaclass=CaseInsensitiveEnumMeta from corehttp.utils, enabling case-insensitive lookup by string value. (`class AppType(str, Enum, metaclass=CaseInsensitiveEnumMeta): STRIPE = 'stripe'`)
**_patch.py escape hatch** — _patch.py is intentionally empty (patch_sdk is a no-op) but must not be deleted. Customizations to generated models go here to survive regeneration. (`# In _patch.py: add helper methods or monkey-patches to generated classes`)

## Key Files

| File | Role | Watch For |
|------|------|-----------|
| `_models.py` | Contains all ~200+ generated dataclass-style model classes. Each extends _Model from _utils/model_base. | DO NOT EDIT — regenerated by make gen-api. Customizations belong in _patch.py. |
| `_enums.py` | All string enums used in models and operations. All use CaseInsensitiveEnumMeta so string comparisons are case-insensitive. | DO NOT EDIT — regenerated by make gen-api. |
| `__init__.py` | Re-exports all models and enums plus applies _patch.py customizations via _patch_sdk(). The __all__ list is the authoritative public API of this subpackage. | DO NOT EDIT — regenerated. New public symbols added in _patch.py must also appear in _patch.__all__. |
| `_patch.py` | The only file safe to hand-edit. Place SDK customizations here that must survive regeneration. | patch_sdk() must remain callable even if empty; do not remove it. |

## Anti-Patterns

- Editing _models.py or _enums.py directly — they are overwritten by make gen-api
- Subclassing generated models outside of _patch.py
- Using plain str Enum without CaseInsensitiveEnumMeta (breaks case-insensitive wire value matching)

## Decisions

- **All models are MutableMapping-backed (_Model) rather than plain dataclasses.** — Enables dict-style access and JSON serialization without a separate to_dict step, matching the corehttp pipeline's expectation for request bodies.
- **_patch.py is preserved across regenerations as the customization point.** — Generated code must remain pristine for automated regeneration; _patch.py provides a stable escape hatch.

## Example: Construct a model with typed kwargs or a raw dict

```
from openmeter._generated.models import Customer
# Typed construction
c = Customer(name='Acme', external_mapping_key='acme-001')
# Raw dict construction (e.g. from parsed JSON)
c = Customer({'name': 'Acme', 'externalMappingKey': 'acme-001'})
```
